# 🐾 DigiPet — Dijital Evcil Hayvan Uygulaması

DigiPet, React Native ve Expo ile geliştirilmiş, 
oyunlaştırma dinamikleri içeren bir dijital evcil hayvan uygulamasıdır.

## 🎮 Özellikler

- 🎰 Slot machine animasyonuyla 15 farklı pet tipi
- 🍖 Besle, 🎮 Oyna, 😴 Uyut, 🛁 Yıka aksiyonları
- ⏰ Zaman bazlı otomatik stat düşüşü
- 🏆 5 seviyeli büyüme sistemi (Yumurta → Efsane)
- 🏅 7 farklı rozet kazanma
- 📋 Günlük görevler ve ödüller
- 🛒 Altınla item satın alınan mağaza
- 🌙 Karanlık mod desteği
- 🌍 Türkçe / İngilizce dil desteği
- 💾 AsyncStorage ile otomatik kayıt

## 🚀 Nasıl Çalıştırılır?

### Gereksinimler
- Node.js 18+
- Expo CLI
- Android Studio (emülatör için)

### Kurulum
```bash
# Repoyu klonla
git clone https://github.com/eraycubukcu/DigiPet.git

# Klasöre gir
cd DigiPet

# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npx expo start
```
